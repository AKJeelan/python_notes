# Variables in Python

## What is a Variable?

Think of a variable as a **labeled box** — you put something inside and refer to it by name.

## Creating Variables

In Python, you create a variable by writing a name, then `=`, then the value. Python figures out the type automatically. No need to declare anything!

### Examples

```python
player_name = "Alex"       # string (text)
score       = 100          # integer (whole number)
speed       = 9.5          # float (decimal)
is_alive    = True         # boolean (True/False)

# Variables in Python

Here's a summary of what each example teaches:

**Example 1 — Game Score** shows that variables can be updated. score = score + 500 takes the old value and replaces it with a new one. This is one of the most common things you'll do in programming.

**Example 2 — Café Order** shows that variables can store numbers and do math. By naming values like coffee_price and tax_rate, your code becomes readable — almost like plain English.

**Example 3 — About Me Card** shows mixing types together. You can have strings, numbers, and booleans all in one program, and combine them into a single output using an f-string.

## Key Rules to Remember

- Use = to assign a value to a variable
- Variable names should be descriptive (player_score is better than x)
- Use underscores for spaces: my_variable, not myVariable (Python convention)
- Variables can be reassigned at any time — they're not fixed!
